Java Coding Test Submission - Router Patch Tester.

A project created to demonstrate some of my Java programming skills. Specification was provided by BT for my industrial year application. Program takes in a CSV file and then outputs to terminal the details of routers that meet the update criteria. CSV is expected to consist of lines containing 5 comma separated values. The contents of these lines are then tested against the update criteria. (Hostname/IP Address have not been repeated, router has not already been patched, version is 12+ and information is logical.)
